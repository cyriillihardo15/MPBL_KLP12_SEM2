About Us

Mini PBL is a derivative of PBL (Project Based Learning). PBL is the use of various kinds of intelligence needed to confront real world challenges, the ability to deal with all that is new and existing complexities. In this Mini PBL we are tasked to build a python web based port scanner application. Our project building time is 1 semester and in this team we have 6 members. To build the app we used python flask framework and bootstrap.

  • Fast & Accurate Scanning

    The python port scanner can do scanning fast and accurate.

  • Responsive UI/UX

    This Website has responsive ui design, and has many compatibility to other devices resolution

  • Open Source

    This python port scanner is free to use and access to all users.
